# xion-types

Generated Rust types for Xion blockchain protocol buffers.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
xion-types = { git = "https://github.com/burnt-labs/xion-types" }
```

Or use a specific version from crates.io:

```toml
[dependencies]
xion-types = "0.1.0"
```
